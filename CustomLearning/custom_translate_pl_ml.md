# <a name="translate-learning-pathways-custom-playlists"></a>Перевод пользовательских плейлистов схем обучения
С помощью путей обучения способ работы с настраиваемой игрой зависит от того, как настроить пути обучения.  

- Если вы создали новый многоязычный сайт путей обучения, вы можете скопировать или создать новый плейлист и перевести список воспроизведения на языки, поддерживаемые сайтом.
- Если вы обновили существующий сайт путей обучения с уже существующими пользовательскими плейлистами, вы можете перевести существующие списки воспроизведения на языки, которые поддерживает ваш сайт. Вы также можете скопировать и создать новые списки воспроизведения. 

> [!IMPORTANT]
> - Предоставленные Майкрософт плейлисты уже переведены и не могут быть изменены, если не создать копию списка воспроизведения. 

## <a name="create-or-copy-a-playlist-and-have-it-translated"></a>Создание или копирование списка воспроизведения и его перевод
Если вы создали новый многоязычный сайт путей обучения или обновили сайт с помощью многоязычной веб-части, вы можете скопировать или создать настраиваемый плейлист и перевести активы плейлистов и плейлистов на языки, поддерживаемые сайтом. 

### <a name="copy-a-playlist-and-have-it-translated"></a>Скопируйте список воспроизведения и переведут его
Чтобы продемонстрировать, как создать новый пользовательский плейлист и перевести его, мы будем использовать функцию copy playlist путей обучения.

1.  Из домашнего меню **"Пути обучения"** щелкните **администрирование путей обучения.** 
2.  В **рамках microsoft 365 пути** обучения выберите Первые дни, а затем выберите шесть простых шагов - Добро пожаловать в Microsoft **365**.  
3. Нажмите **кнопку Скопируйте список воспроизведения,** прокрутите страницу и нажмите **кнопку Изменить подробно**.   
4. Измените название списка воспроизведения. В этом примере мы изменим название списка воспроизведения на **пять простых действий.** Доступные языки будут зависеть от поддерживаемых языков, выбранных для сайта. Чтобы добавить дополнительные языки, необходимо добавить на сайт дополнительные поддерживаемые языки.   
5. Нажмите **кнопку Сохранить подробно**. 
6. Щелкните язык справа от языка по умолчанию (в данном случае английский), чтобы просмотреть скопированные сведения о списке воспроизведения. Сведения для нового языка отражают исходный скопированные плейлисты и требуют перевода.
7. Обратите внимание, чтобы уведомить переводчика о переводе сведений из списка воспроизведения.  

![Пример итальянской страницы.](media/custom_update_ml_pldetail.png)

> [!IMPORTANT]
> Уведомления переводчика не встроены в настраиваемые списки воспроизведения. Переводчики должны быть уведомлены вручную. Переводчики вручную переводят копии языковых сведений по умолчанию на указанный язык(ы). Необходимо уведомить переводчика о переводах, необходимых для сведений о списке воспроизведения. Рекомендуется завершить все изменения списка воспроизведения, включая добавление, редактирование или удаление активов плейлиста, а затем уведомить переводчика о необходимых переводах для сведений и активов плейлиста.

## <a name="create-multilingual-assets-in-a-custom-playlist"></a>Создание многоязычных активов в настраиваемом списке воспроизведения
Любые активы в списке воспроизведения, предоставленные Корпорацией Майкрософт, не являются редактируемыми и не требуют перевода. Предоставленные Корпорацией Майкрософт активы будут доступны на языках, поддерживаемых сайтом. Для настраиваемых активов, которые вы создаете, или существующих настраиваемых активов, можно добавить языки, поддерживаемые сайтом.  Рассмотрим, как добавить языки для новых и существующих настраиваемой информации в плейлист. 

### <a name="create-a-new-multilingual-asset-for-a-playlist"></a>Создание нового многоязычного актива для списка воспроизведения
В этих инструкциях предполагается, что в этом примере используется плейлист Five Simple Steps. 

**Создание актива**
1. В списке пять простых действий в **статье Assets** выберите **+ Новый** актив и заполните сведения для актива. 
   - Для **заголовка** введите "Добро пожаловать в Microsoft 365 в Contoso". 
   - Для **технологии** выберите **Microsoft 365**.
![Форма, в которой вы вводите сведения об активах.](media/custom_ml_pl_createasset.png)
2. Выберите **Создание актива,** а затем выберите **Сохранить актив**.

**Открытие актива**
1. Выберите созданный актив и нажмите кнопку **Открыть**.  
2. Внести изменения на страницу по желанию. В настоящее время можно изменить название страницы и добавить текст на страницу, как показано в следующем примере, а затем нажмите **кнопку Опубликовать**. 
3. Нажмите **кнопку Перевод**.
4. Выберите **Создать** для языка(ы) вы хотите. В этом примере мы выбираем итальянский язык.
5. Выберите **Представление**. Теперь вы должны видеть страницу, аналогичную следующему примеру, в зависимости от поддерживаемых языков.
![Пример страницы Активы плейлиста.](media/custom_ml_pl_createasset.png)
6. Нажмите кнопку **Опубликовать**.
7. На созданной странице перевода скопируйте URL-адрес из браузера и вернись на страницу администрирования путей обучения.

**Добавление языка для созданного средства перевода** 
1. Вернись на страницу администрирования путей обучения и выберите значок Редактирование (карандаш) для открываемой вами актива "Пять простых шагов". 
2. Нажмите **кнопку Добавить язык,** а затем выберите язык. В этом примере мы выбрали итальянский язык. 
3. Введите URL-адрес созданной страницы с переводом. Вы должны увидеть страницу, аналогичную следующему примеру. 
![Форма, в которой вы вводите URL-адрес определенной языковой страницы](media/custom_ml_pl_translateasseturl.png)
4. Нажмите **кнопку Сохранить актив**.

## <a name="add-languages-for-existing-custom-playlists-and-assets"></a>Добавление языков для существующих пользовательских списков воспроизведения и ресурсов
Если вы обновили пути обучения для многоязычной поддержки и имеете существующие настраиваемые списки воспроизведения и ресурсы, вы можете добавить поддерживаемые языки для списков воспроизведения и активов. Эти инструкции предполагают, что у вас есть настраиваемый плейлист, к которому можно добавить языки. Если у вас нет пользовательского списка воспроизведения, см. в этой теме раздел "Создание или копирование списка воспроизведения". 

> [!IMPORTANT]
> - Предоставленные Майкрософт плейлисты уже переведены и не могут быть изменены, если не создать копию списка воспроизведения. Можно добавлять только языки для настраиваемой воспроизведения. 
> - Чтобы добавить языки для пользовательских списков воспроизведения и ресурсов, необходимо включить многоязычную поддержку и языки. Дополнительные сведения см. в [дополнительных сведениях в set Language Settings.](./custom_update_ml.md#set-language-settings) 

### <a name="to-add-languages-for-an-existing-playlist"></a>Добавление языков для существующего списка воспроизведения
1.  Из домашнего меню **"Пути обучения"** щелкните **администрирование путей обучения.** 
2.  Выберите настраиваемый список воспроизведения. Пользовательский список воспроизведения идентифицирован значком people с помощью контрольного знака, как показано в следующем примере.
![Пример пользовательского списка воспроизведения.](media/custom_ml_pl_customselect.png) 
3. Выберите **Изменение детализации**. Может потребоваться прокрутка вниз, чтобы найти кнопку **Изменить детализацию.**   
4. Выберите **Добавить язык,** а затем выбрать язык.  
5. Выберите **сохранить деталь**.
 Доступные языки будут зависеть от поддерживаемых языков, выбранных для сайта. Чтобы добавить дополнительные языки, необходимо добавить на сайт дополнительные поддерживаемые языки.   
5. Щелкните язык справа от языка по умолчанию (в данном случае английский), чтобы просмотреть скопированные сведения о списке воспроизведения. Сведения для нового языка отражают исходный скопированные плейлисты и требуют перевода.
![Пример страницы сведений для итальянского языка.](media/custom_ml_pl_editdetailit.png) 
7. Обратите внимание, чтобы уведомить переводчика о переводе сведений из списка воспроизведения.  

**Открытие актива**
1. Выберите созданный актив и нажмите кнопку **Открыть**.  
2. Внести изменения на страницу по желанию. В настоящее время можно изменить название страницы и добавить текст на страницу, как показано в следующем примере, а затем нажмите **кнопку Опубликовать**. 
3. Нажмите **кнопку Перевод**.
4. Выберите **Создать** для языка(ы) вы хотите. В этом примере мы выбираем итальянский язык.
5. Выберите **Представление**. Теперь вы должны видеть страницу, аналогичную следующему примеру, в зависимости от поддерживаемых языков.
![Пример страницы, которая отображается при выборе Представления.](media/custom_ml_pl_createasset.png)
6. Нажмите кнопку **Опубликовать**.
7. На созданной странице перевода скопируйте URL-адрес из браузера и вернись на страницу администрирования путей обучения.

## <a name="notify-the-translator"></a>Уведомление переводчика
Когда вы закончите создание средств перевода, уведомите переводчика о необходимых переводах. Переводчик:
- Перевод сведений о списке воспроизведения.
- Перевод сведений об активах.
- Перевод добавленных языковых страниц для актива.
- Уведомите запросителя о том, что переводы готовы к проверке

## <a name="add-languages-for-an-existing-custom-playlist"></a>Добавление языков для существующего пользовательского списка воспроизведения
Если вы обновили пути обучения для многоязычной поддержки и имеете существующие настраиваемые списки воспроизведения и ресурсы, вы можете добавить поддерживаемые языки для списков воспроизведения и активов. 

> [!IMPORTANT]
> - Предоставленные Майкрософт плейлисты уже переведены и не могут быть изменены, если не создать копию списка воспроизведения. Можно добавлять только языки для настраиваемой воспроизведения. 
> - Чтобы добавить языки для пользовательских списков воспроизведения и ресурсов, необходимо включить многоязычную поддержку и языки. Дополнительные сведения о включаемой многоязычной поддержке и параметрах см. в [материале Set Language Settings.](./custom_update_ml.md#set-language-settings) 


## <a name="instructions-for-the-translator"></a>Инструкции для переводчика
При добавлении языков в плейлисты необходимо уведомить переводчика об изменениях. Вы можете указать переводчику следующее 

### <a name="translate-playlist-details"></a>Перевод сведений о списке воспроизведения
Из домашнего меню "Пути **обучения"** нажмите **кнопку Администрирование путей обучения.** 
1. Щелкните настраиваемый список воспроизведения, требуемый для перевода, а затем нажмите языки. 
2. Нажмите **кнопку Изменить подробно,** сделать переводы для списка воспроизведения, а затем нажмите кнопку 
3. Нажмите **кнопку Сохранить подробно**. 
4. Уведомите запрашивателя перевода о том, что перевод завершен. 

### <a name="translate-asset-details"></a>Перевод сведений об активах
Из домашнего меню "Пути **обучения"** нажмите **кнопку Администрирование путей обучения.** 
1. Щелкните настраиваемый список воспроизведения, требуемый для перевода. 
2. Прокрутите страницу вниз, затем в статье Assets выберите изменение для актива, который необходимо изменить, а затем выберите язык. 
3. Сделайте переводы для актива и нажмите кнопку **Сохранить актив**.  

### <a name="translate-the-added-language-page-for-the-asset"></a>Перевод добавленной языковой страницы для актива
Из домашнего меню "Пути **обучения"** нажмите **кнопку Администрирование путей обучения.** 
1. Щелкните настраиваемый список воспроизведения, требуемый для перевода. 
2. Прокрутите страницу, затем в статье Assets выберите актив, выберите язык и нажмите кнопку Открыть. 
3. Сделайте переводы для страницы и нажмите кнопку **Опубликовать**.
