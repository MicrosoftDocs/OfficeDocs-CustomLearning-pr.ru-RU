# <a name="microsoft-365-learning-pathways-content-updates-for-april"></a>Обновления контента microsoft 365 для учебных путей за апрель
Пути обучения стремятся придерживаться циклов обновления ежеквартального контента. В следующем списке обновления контента за апрель 2020 г.

## <a name="april-2020-content-updates"></a>Обновления контента за апрель 2020 г.
 В связи с текущими событиями обновление контента за апрель 2020 г. предоставляет списки обучающих игр и ресурсы, предназначенные для удаленной работы в организации. Мы добавили новый набор удаленных плейлистов Work, значительно обновили контент Microsoft Teams, особенно для собраний Teams, добавили новый набор Office для веб-плейлистов и добавили новые активы Microsoft Forms. В рамках обновления мы также предоставили список ресурсов удаленной работы для администраторов.  
 
 Дополнительные материалы о применении обновлений контента и управлении обновлениями контента см. в материалах [Manage Content Updates.](custom_contentupdatesmanage.md) В следующих списках перечислены обновления контента для microsoft 365 в апреле 2020 г.
